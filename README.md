# GDF 2018 - Web Components
Working example for Web Components talk at Google Devfest 2018
This is a basic to advanced walkthrough of creating Web Components with Angular 2+
but can be applied to any other frameworks. Web components are giving engineers a
new way of writing, managing, scaling, and testing UI.

Running the following command will compile your typescript and run gulp sass

```javascript
npm install
npm start
localhost:3000

NOTE: node v6.0.0 is required
```

The application contains all the contents for the talk. There is a route of for
all the steps to create the applications components
```javascript
localhost:3000/steps
```
